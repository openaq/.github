
# Contribution guide

OpenAQ welcomes external contribution to public open-source repositories.
Contributors must follow and abide by this contribution guide and repository
specific guidelines such as CONTRIBUTING.md files, issue templates, pull request
templates.

Contributions are defined as:

- Issues: bug reports and feature requests
- Discussions: support questions and feature improvement topics
- Code: Submitting pull requests for repository contents including docuemntation and code.

To maintain a healthy, respectful and sustainable project, all contributions must abide
by the OpenAQ [Community Code of Conduct](https://github.com/openaq/.github/blob/main/CODE_OF_CONDUCT.md) and the
[LLM usage policy](https://github.com/openaq/.github/blob/main/LLM_POLICY.md).
 
OpenAQ reserves the right to delete contribution content that does not
follow contribution guidelines and in extreme cases block contribution if
persistent violation of code of conduct or failure to follow contribution
guidelines. These guidelines are subject to change at the discretion of the
OpenAQ staff.
